# 문서 템플릿

OpenTelemetry 운영 문서를 빠르게 만들기 위한 템플릿입니다. 실제 서비스 문서, 런북, 장애 보고서를 작성할 때 복사해서 사용합니다.

## 문서 목록

| 문서 | 용도 |
|------|------|
| [service-doc.md](service-doc.md) | 서비스별 OpenTelemetry 계측 현황 정리 |
| [runbook.md](runbook.md) | 알림 발생 시 확인/진단/조치 절차 정리 |
| [incident-report.md](incident-report.md) | 장애 타임라인, 영향, 원인, 재발 방지 정리 |

## 사용 기준

| 작업 | 사용할 템플릿 |
|------|------|
| 새 서비스를 계측한 뒤 문서화 | [service-doc.md](service-doc.md) |
| 알림 대응 절차 작성 | [runbook.md](runbook.md) |
