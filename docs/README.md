# OpenTelemetry Docs

이 디렉터리는 목적별 번호 폴더로 문서를 관리합니다. 앞의 번호는 권장 학습 및 운영 참조 순서를 나타냅니다.

## 문서 구조

| 폴더 | 내용 |
|------|------|
| `01-installation/` | OpenTelemetry 설치, 배포, 업그레이드 절차를 다룹니다. |
| `02-architecture/` | OpenTelemetry 아키텍처와 데이터 흐름을 다룹니다. |
| `03-signals/` | Metrics, Logs, Traces 등 telemetry signal의 개념과 사용 기준을 다룹니다. |
| `04-instrumentation/` | 애플리케이션 계측과 SDK/auto instrumentation을 다룹니다. |
| `05-collector/` | OpenTelemetry Collector 구성, pipeline, receiver, processor, exporter를 다룹니다. |
| `06-practice/` | 실습과 end-to-end 예제를 다룹니다. |
| `07-operations/` | 운영, 모니터링, 장애 대응, 성능 기준을 다룹니다. |
| `90-standards/` | 문서 작성과 OpenTelemetry 운영 규칙을 다룹니다. |
| `91-templates/` | 재사용 문서 템플릿을 둡니다. |
| `99-agents/` | AI 작업 보조 프롬프트를 둡니다. |

실행 가능한 예제와 운영 보조 자료는 [../ops/README.md](../ops/README.md)를 참고합니다.
