# AI 작업 지침

이 디렉터리는 OpenTelemetry 문서를 다룰 때 사용하는 AI 작업 지침을 모은다.

## 문서 목록

| 문서 | 용도 |
|------|------|
| `doc-writer.md` | 문서 작성 기준 |
| `collector-designer.md` | Collector 파이프라인 설계 |
| `instrumentation-advisor.md` | 계측 전략 |
| `troubleshooter.md` | 문제 해결 절차 |

## 언제 보나

- 새 문서를 쓸 때 `doc-writer.md`
- Collector 파이프라인을 설계할 때 `collector-designer.md`
- 자동/수동 계측을 정리할 때 `instrumentation-advisor.md`
- 장애 분석이나 재현이 필요할 때 `troubleshooter.md`
